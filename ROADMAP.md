# Roadmap

## Phase 1: Foundation
- [ ] OS-level Accessibility API integration (macOS focus).
- [ ] monkey-see: Basic screen recording and event capture (clicks, keyboard).
- [ ] Core project structure and monorepo setup.

## Phase 2: Skill Extraction
- [ ] Event stream to structured "Step" conversion.
- [ ] Vision-augmented event validation.
- [ ] Skill serialization format.

## Phase 3: monkey-do (Execution & Replay)
- [ ] Deterministic replay engine.
- [ ] Adaptive replay (handling UI shifts/latency).
- [ ] Intent-based step validation.

## Phase 4: The Agent
- [ ] Natural language to skill mapping.
- [ ] Open-ended goal planning using recorded skills as primitives.
- [ ] Feedback loops and self-correction during execution.
