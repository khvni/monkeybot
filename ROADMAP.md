# Roadmap

## Phase 1: The Core (monkey-see & monkey-do)
- [ ] OS-level Accessibility API integration (macOS focus).
- [ ] **monkey-see**: Basic screen recording and event capture (clicks, keyboard).
- [ ] **monkey-do**: Core agent harness for taking actions and replaying demonstrations.
- [ ] Core project structure and monorepo setup.

## Phase 2: Voice & Interaction (monkey-say)
- [ ] **monkey-say**: Implement realtime voice interaction layer.
- [ ] Voice-to-intent mapping for agent control.
- [ ] Feedback loops for teaching tasks via voice.

## Phase 3: Advanced Agent Capabilities
- [ ] Integration with Agent Client Protocol (ACP) for dispatching agents.
- [ ] Coding agent primitives: file editing, terminal commands, and generation.
- [ ] Deterministic and adaptive replay for demonstrations.

## Phase 4: Intelligence & Autonomy
- [ ] Natural language/voice to skill mapping.
- [ ] Open-ended goal planning using recorded skills as primitives.
- [ ] Self-correction and vision-augmented validation during execution.
